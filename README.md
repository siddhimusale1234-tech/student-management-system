# Student Management System

A simple console-based Student Management System built with Python.

## Features

- Add Student
- View Students
- Search Student
- Delete Student
- Prevent Duplicate Entries
- Input Validation
- Student List in Alphabetical Order
- Total Student Count
- Modern Python `match-case` Implementation

## Technologies Used

- Python 3.10+
- Match-Case Statement
- Functions
- Lists

## How to Run

1. Clone the repository

```bash
git clone https://github.com/prajwalkarajange/student-management-system.git
```

2. Open the project

```bash
cd student-management-system
```

3. Run the program

```bash
python student_management.py
```
